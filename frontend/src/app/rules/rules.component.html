OYE OYE LA BATAILLE ROYALE REPREND DU SERVICE

Pour le mois de février (1er au 28 février) nous vous lançons un nouveau défi pour vous garder en forme ! Les choses s’accélèrent et les activités reprennent… alors nous vous proposons de bien vous échauffer pour attaquer cette nouvelle année (ou pour éliminer les excès de la précédente )!

2 équipes seront en compétition (2 canaux Slack seront créés)

  • Équipe Internationale avec comme capitaine @Carl & capitaine adjoint @Jessica (personnes travaillant ailleurs que Montréal ou pour nos clients à l’international)

VS

  • Équipe Montréal avec comme capitaine @Bernard & capitaine adjoint @Rose Dansereau (personnes travaillant à Montréal)

L’objectif: bouger pour la bonne cause…

1h d’activité sportive = 1$ reversé à une association par Techso.

L’équipe qui aura cumulé le plus d’heures dans le mois choisira l’association à laquelle reversé les $$$ (max 500$ - soit 500h).

Modalités

  1. Chaque équipe devra choisir l’association qu’elle souhaite soutenir.
  2. Un minimum de 15min par activité est requis.
  3. Toutes les activités sportives comptent : …
  4. Comment comptabiliser vos min/heures? Via votre montre connectée ou l’application Strava* est le plus simple.
  5. Un club Strava Techso est disponible pour voir les activités de chacun (on vous ajoute sur demande).
  6. Nous avons besoin d’une preuve de votre activité : le plus simple est de rejoindre le groupe Techso sur Strava.
    • Si vous n’êtes pas à l’aise avec l’idée envoyez-nous (Rose / Jess) une capture d’écran de votre activité, une photo par exemple…
    • Sinon, vous pouvez importer vos activités d'aujourd'hui En utilisant la page prévue à cet effet
  7. Un suivi des points sera disponible avec des mises à jour régulières Ici

*Pour utiliser strava

  1. Il faut créer un compte (gratuit)
  2. Ajoutez Jess (Jessica Glnd) et/ou Rose (Rose Dansereau) pour que l’on vous ajoute sur le Club Strava Techso.
  3. Vous pourrez ensuite synchroniser votre montre connectée et vos activités se mettront automatiquement dans l’application.
    • Si vous n’avez pas de montre vous pouvez lancer votre activité directement avec l’application Strava (multitude d'activités disponibles sur l’application : etc vous devrez trouver votre bonheur).
    • Si vous n’avez pas pu lancer votre montre, Strava, ou que votre activité n’est pas dans la liste, vous pouvez saisir manuellement votre activité dans Strava (là une preuve sera exigée).
